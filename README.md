# **📊 Coders of Delhi**  
A beginner-friendly collection of Python data science mini-projects built using **Pure Python** and the  **JSON module**.  
These projects were created after learning the concepts from **CodeWithHarry's Data Science Course**. The goal was to first understand the concepts and then implement them independently through hands-on projects.  
## **🚀 Projects**  
### **1. Data Formatting**  
- Read and process JSON data.  
- Format and organize raw data into a structured form.  
- Improve readability and consistency.  
### **2. Data Cleaning**  
- Handle missing or invalid values.  
- Remove duplicate entries.  
- Clean and prepare data for further analysis.  
### **3. People You May Know**  
- Simulates a friend recommendation system.  
- Uses JSON data to identify potential connections based on shared information.  
### **4. Pages You Might Like**  
- A simple recommendation project that suggests pages based on user interests stored in JSON files.  
## **🛠️ Technologies Used**  
- Python 3  
- JSON Module  
- File Handling
